# radium - WPF Telerik Extensions
A collection of WPF class libraries extending Telerik UI for WPF controls. Implements common patterns (like MVVM) and custom functionality to streamline application development and ensure consistent UI/UX across projects.

## Overview

This repository contains custom WPF Class Libraries designed to enhance and standardize the use of Telerik UI for WPF controls within our development ecosystem.

It implements mine preferred MVVM pattern to provide a clean, maintainable, and reusable set of components that are ready for immediate use in new and existing WPF applications.

Key Features

Pattern Implementation: Provides base classes and helpers that enforce our specific MVVM implementation for Telerik controls (e.g., custom ViewModelBase, extended RadWindow commands).

Extended Controls: Custom wrappers and styles for popular Telerik controls (e.g., RadGridView, RadRibbonView) with pre-configured settings for a standard look-and-feel.
